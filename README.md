# DATA_221_Assignment_1

DATA_221_Assignment_1_Question_1:
A program that finds the smallest factorial that exceeds the value of a given threshold number.

DATA_221_Assignment_1_Question_2:
A function that returns a dictionary storing the length and parity of each string in a given list.

DATA_221_Assignment_1_Question_3:
A function that finds the power of the first number to the second number from a list of pairs.

DATA_221_Assignment_1_Question_4:
A fanction chooses a random float from 0 to 1 and a randomly long list of randomly generated floats from 0 to 1 and finds the first index that is equal to or greater than the random float.

DATA_221_Assignment_1_Question_5:
A function that takes input for 2 radii and compares the areas of the circles that the radii form, then returns the percentage of the larger circle's area that the smaller circle can cover.

DATA_221_Assignment_1_Question_6:
A function that takes a list and returns a sorted dictionary with the keys being the values of the list and the values representing the percentage of the other pairs that the key is larger than.

DATA_221_Assignment_1_Question_7:
A function that takes input for a number of seconds and returns the number of hours, minutes and seconds since midnight it would be, if any, as well as if it results in AM or PM.

DATA_221_Assignment_1_Question_8:
A function that takes a dictionary of columns and adds a new column by adding the values in columns 'A' and 'B' then using the resulting valuesd as a divisor for column 'C', then turns the data into a data frame and prints the result.
